PricePoint Developer Bootstrap

PricePoint dev machine setup

Install latest version of VirtualBox

Download latest stable version of Lubuntu

Create new VM with at least 200gb disc, 4 cores, 16gb ram, and 64mb video memory. Be sure to also update clipboard and drag and drop settings to bidirectional.

Detailed Procedure:

  1. Create a new VM. Assign 8GB Memory and at least 200GB Disc.
  2. Go to Settings. Under General > Advanced, Change the Clipboard and Drag and Drop settings to bidirectional
  3. Under Storage, select the Empty CD drive. Click the CD Icon next to the drop down and select Choose a disk file. Select the Lubuntu iso
  4. Start your VM and run the OS installer on the desktop. Let it complete then shutdown the VM.
  5. Go back to where you chose the Lubuntu image, this time select Remove Disk from Virtual Drive
  6. Increase the VM resources. It should have at least 4 cores, 16GB ram, and 64mb video memory

Install Guest Additions

  1. Start up your VM. In the menu bar select Devices > Insert Guest Additions CD Image
  2. In a terminal, run:
    sudo apt-get install -y dkms build-essential linux-headers-generic linux-headers-$(uname -r)
  3. Navigate to the CD directory in your terminal. It should be /media/{username}/VBox_GAs_X.X.XX
  4. Change to root user with:
    sudo su
  5. Install Guest Additions with the command:
    ./VBoxLinuxAdditions.run
  6. Restart VM

Run the following in a terminal window and follow the remaining prompts

wget -qO bootstrap.sh https://api.github.com/repos/Price-Point/development-bootstrap/contents/bootstrap.sh --header 'Accept: application/vnd.github.v3.raw' && bash bootstrap.sh && rm bootstrap.sh

Further setup

  1. Follow instructions here for setting up GitHub SSH Keys
  2. Setup local mysql root password by running sudo mysql and then update the following command with the right password (ask Phoenix) and run it. ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'new_password_here';
